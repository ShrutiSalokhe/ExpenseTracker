# ExpenseTracker
A Java Swing-based desktop application for tracking daily expenses, managing budgets, categorizing spending, and monitoring expenses with an intuitive GUI.
# 💰 Daily Expense Tracker

A simple and user-friendly desktop application built using **Java Swing** that helps users manage their daily expenses, set budgets, and monitor spending efficiently.

---

## 📌 Features

- 📅 Add expenses with date
- 💵 Set a spending budget
- 📂 Categorize expenses (Food, Shopping, Travel, Study, Other)
- 📝 Add notes for each expense
- 📊 View category-wise expense summary
- ⚠️ Budget exceeded warning
- 🗑️ Delete the last added expense
- 🔄 Clear all entered data
- 🎨 Multiple themes (Light, Dark, Blue)
- 💰 Real-time total expense calculation

---

## 🛠️ Technologies Used

- Java
- Java Swing
- AWT
- Event Handling
- Object-Oriented Programming (OOP)
